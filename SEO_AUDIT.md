# SEO-аудит сайта cryptobot.ltd

## Краткий вывод
Сайт уже имеет базовую техническую SEO-структуру (canonical, hreflang, robots.txt, sitemap.xml), но для выхода в ТОП-1 Google этого недостаточно: сейчас не хватает глубины контента, внешнего авторитета домена и полноценной контентной воронки под поисковые запросы.

## Что уже хорошо
- Есть отдельные языковые страницы: EN / RU / ZH.
- Прописаны `canonical` и `hreflang`.
- Есть `robots.txt` и `sitemap.xml`.
- Контент доступен без JavaScript (статический HTML).

## Что было исправлено в этом коммите
1. Исправлена ошибка в переключателе языка на главной странице (битый `value` у English-опции).
2. Добавлены метатеги индексации и сниппетов: `robots`, `theme-color`.
3. Улучшены Open Graph / Twitter Card метаданные.
4. Добавлена структурированная разметка Schema.org (`Organization`, `WebSite`) на всех языковых страницах.
5. Добавлены `aria-label` для языкового селектора (доступность).
6. Добавлены `rel="noopener noreferrer"` для внешних ссылок с `target="_blank"`.
7. Обновлён `sitemap.xml`: синхронизированы alternate-ссылки для всех языков.

## Что нужно сделать, чтобы реально бороться за ТОП-1

### 1) Семантика и контент (самый высокий приоритет)
- Сделать полноценные SEO-лендинги под кластеры запросов, например:
  - «crypto bot telegram»,
  - «buy crypto in telegram»,
  - «telegram p2p crypto market»,
  - RU: «криптобот телеграм», «купить крипту в телеграм»,
  - ZH-локальные варианты.
- Добавить разделы:
  - FAQ (с разметкой FAQPage),
  - Guides / Help Center,
  - Безопасность и комплаенс,
  - Тарифы/комиссии,
  - Страницы отдельных функций (P2P, Exchange, Crypto Pay, Checks, Invoices).

### 2) EEAT и доверие
- Отдельные страницы «О компании», «Контакты», «Юридическая информация», «Политика конфиденциальности», «Условия использования».
- Показать команду, даты обновления материалов, источники, прозрачность процессов.
- Укрепить брендовые сигналы (упоминания в медиа, профили, верифицированные соцсети).

### 3) Техническая оптимизация
- Перейти с `.html` на чистые локализованные URL (`/ru/`, `/zh/`) — по возможности.
- Добавить `WebPage`, `BreadcrumbList`, FAQ schema там, где уместно.
- Проверить Core Web Vitals в PageSpeed Insights и Search Console.
- Настроить HTTP-заголовки кеширования для CSS/изображений.

### 4) Внешняя оптимизация (Off-page)
- Получить тематические обратные ссылки:
  - крипто-СМИ,
  - Telegram-каталоги,
  - обзоры кошельков/ботов,
  - партнёрские интеграции.
- Digital PR: релизы обновлений продукта + гостевые публикации.

### 5) Аналитика и итерации
- Подключить Google Search Console и GA4 (если ещё не подключены).
- Отслеживать:
  - индексируемые страницы,
  - CTR по запросам,
  - позиции по кластерам,
  - конверсию в старт бота.
- Каждые 2–4 недели обновлять страницы, повышая полноту и полезность.

## Важное ожидание
ТОП-1 нельзя гарантировать «одним правками по коду». Это результат стратегии на 3–6+ месяцев: контент + ссылки + бренд + техническое качество + поведенческие сигналы.
